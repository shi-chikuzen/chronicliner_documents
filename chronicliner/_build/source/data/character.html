キャラクター設定

Excelファイルの キャラクター シートを使用し、各キャラクターの設定を行います。

1. 概要

設定例

誕生年自動計算

必須

TRUE 、 FALSE

学年・年から誕生年を逆算するか

TRUE

注釈

誕生日自動計算を使用する場合、誕生日に設定した年数は無視されます

警告

誕生日自動計算を使用する場合、教育課程設定 において 起算年・起算学年 の設定が必要になります

2. 各設定項目

キャラクタ名

キャラクターの名前を設定します。

警告

カテゴリとキャラクターには同一名称を使用できません

カテゴリ

キャラクターの所属するカテゴリを指定します。

警告

カテゴリ設定 で設定していないカテゴリ名を指定するとエラーが発生します

誕生日

キャラクターの誕生日を yyyy-MM-dd 形式で設定します。

誕生年自動計算 を使用する場合、西暦部分に設定された値は無視されるため、適当な値を入力すれば問題ありません。

死亡日

キャラクターの死亡日を yyyy-MM-dd 形式で設定します。

誕生日詳細

キャラクターの誕生日イベントの詳細を設定します

死亡日詳細

キャラクターの死亡日イベントの詳細を設定します。

誕生年自動計算

TRUE を指定した場合、 教育課程設定 の 起算年・起算学年 から、 yyyy 年の教育課程開始月日に n 学年目に進学するとした場合の誕生年を自動計算します。
